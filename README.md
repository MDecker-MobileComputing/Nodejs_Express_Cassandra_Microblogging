# Cassandra mit Nodejs: Mikroblogging #

<br>

Dieses Repositoy enthält ein Nodejs-Projekt, das einen einfachen
[Mikroblogging](https://de.wikipedia.org/wiki/Mikroblogging)-Dienst
mit der verteilten Datenbank [Cassandra](https://cassandra.apache.org/_/index.html)
implementiert.
